# 🐳 Kubernetes Docker 모니터링 서버 (Kubernetes Docker Monitoring Server)

> **2025 한이음 드림업 프로젝트** | **컨테이너 오케스트레이션 모니터링 시스템**

Kubernetes 클러스터와 Docker 컨테이너를 실시간으로 모니터링하고 관리하는 웹 기반 모니터링 시스템입니다. FastAPI 서버와 웹 인터페이스를 통해 컨테이너 상태, 리소스 사용량, 로그 등을 통합 관리할 수 있습니다.

## 🎯 프로젝트 개요

본 프로젝트는 Kubernetes 클러스터와 Docker 컨테이너 환경을 효율적으로 모니터링하고 관리하기 위한 통합 모니터링 시스템입니다. 실시간 데이터 수집, 시각화, 알림 기능을 통해 인프라 상태를 쉽게 파악하고 문제를 신속하게 해결할 수 있도록 지원합니다.

## 🛠 기술 스택

### Backend

- **Python 3.11+** - 메인 프로그래밍 언어
- **FastAPI** - 고성능 웹 프레임워크
- **SQLAlchemy** - ORM 및 데이터베이스 관리
- **MySQL** - 관계형 데이터베이스
- **Pydantic** - 데이터 검증 및 직렬화

### Frontend

- **HTML5/CSS3** - 웹 인터페이스
- **JavaScript (ES6+)** - 클라이언트 사이드 로직
- **Vanilla JavaScript** - 프레임워크/라이브러리 없이 순수 JS 사용
- **Bootstrap** - 반응형 UI 프레임워크
- **Chart.js** - 데이터 시각화
- **Font Awesome** - 아이콘 라이브러리

### Infrastructure & Monitoring

- **Kubernetes** - 컨테이너 오케스트레이션
- **Docker** - 컨테이너화 플랫폼
- **Prometheus** - 메트릭 수집 및 저장

### Development Tools

- **Git** - 버전 관리

## 🚀 주요 기능

### 1. 📊 실시간 모니터링 대시보드

- **컨테이너 상태** 실시간 모니터링
- **리소스 사용량** (CPU, 메모리, 디스크) 추적
- **네트워크 트래픽** 분석
- **시스템 메트릭** 시각화

### 2. 🐳 Docker 컨테이너 관리

- **컨테이너 목록** 조회 및 관리
- **컨테이너 시작/중지/재시작** 기능
- **컨테이너 로그** 실시간 조회
- **컨테이너 상태** 모니터링

### 3. ☸️ Kubernetes 클러스터 관리

- **노드 상태** 모니터링
- **파드(Pod) 관리** 및 상태 추적
- **서비스 및 인그레스** 관리
- **네임스페이스** 관리

### 4. 📈 성능 분석 및 통계

- **시스템 성능** 지표 분석
- **트렌드 분석** 및 예측
- **리소스 사용률** 통계
- **알림 및 이벤트** 관리

### 5. 🔐 사용자 관리 시스템

- **사용자 인증** 및 권한 관리
- **관리자 페이지** 기능
- **로그인/로그아웃** 세션 관리
- **사용자 활동** 로깅

### 6. 📋 로그 관리

- **시스템 로그** 수집 및 분석
- **애플리케이션 로그** 추적
- **오류 로그** 모니터링
- **로그 검색** 및 필터링

## 🐛 문제 해결

### 빠른 해결 방법

- **서버 시작 문제**: 가상환경 활성화 및 의존성 재설치
- **데이터베이스 연결**: MySQL 서비스 상태 확인
- **포트 충돌**: 다른 포트로 실행 (`--port 8001`)
- **인증 오류**: 세션 토큰 및 SECRET_KEY 확인
